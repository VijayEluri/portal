Simply use different background images and change the background colour hex of the .bl rule in order to change the colour of the speech bubble! Please note that the images cannot be tansparent or else they'll simply take on the background colour of the .bl div and the rounded edges will not be visible. Also, feel free to change the width of the speech bubbles by changing the value of the width propery of the .sbl div.
